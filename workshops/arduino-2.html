Arduino 2

Overview

A motion-control workshop covering embedded code and electromechanical systems: safely operating DC and servo motors using PWM, transistors, and external power sources while managing speed and position.

Covered

  • Motor control basics — using servos and motors with Arduino
  • PWM speed regulation to change a motor's speed
  • Using a potentiometer (analog input) to set servo position alongside a digital input
  • Building a circuit to digitally trigger a motor to start spinning

Focus

Understanding system limits, interpreting signals, and converting software logic into physical motion.
